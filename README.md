# Linknau Developer Test Case - Article Feature

## Description
This project implements an article feature for PT ABC's landing page using Golang, Gorm, and PostgreSQL.  
Each article can have multiple photos.

## How to Run
1. Copy the example environment file:
```bash
cp .env.example .env
 ```

 1. Run the application:
```bash
go run cmd/app/main.go
 ```
